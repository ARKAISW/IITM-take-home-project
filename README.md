# Personal Finance Risk Analysis

IITM Programme — Intro to Data Science and AI (Take Home Project)

Analyzes financial risk patterns in a dataset of 200 individuals using EDA, K-Means clustering and Decision Tree classification.

## How to run

```
pip install pandas numpy matplotlib seaborn scikit-learn
python solution.py
python plots.py
```

Figures are saved to `figures/`.
